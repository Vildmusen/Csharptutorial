Steg 1: Nytt projekt

  • Gör ett nytt projekt i Visual studio: Välj "Windows Forms App (.NET Framework)"

  • Dubbelklicka på "form1" som öppnas så kommer ni till koden för den. Det första vi ska göra är att lägga till grafik så vi kan se våran orm.

  • Detta är inget ni har gjort tidigare, så ni får koden rakt av:

  • "InitializeComponent()" Står redan där, men lägg till "IGame game", som kommer vara vårt interface, och "Graphics Drawing" och "Drawing = this.CreateGraphics()" som ska sköta grafiken

Steg 2: Game Interface

  • Skapa ett interface. (Project -> Add class, byt från klass till interface) Ge det namnet "iGame".

  • Ett interface kanske inte är 100% nödvändigt i det här fallet, men bra att fräscha upp minnet om. (Och om vi skulle vilja lägga till fler spel senare!)

  • Låt det vara tomt som det är, vi kommer tillbaka senare när vi har mer koll på vad som behöver vara här!
